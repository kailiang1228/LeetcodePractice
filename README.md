# LeetcodePractice

這裡記錄了我從零開始挑戰 LeetCode 上的練習解答(我弱弱的解答，勿參考)
目標是透過 Python 建立解題直覺，再透過 C 語言實作來精進記憶體管理與位元運算能力
目標能成為韌體工程師

## 當前學習重點
- **Bit Manipulation**: 熟悉 `&`, `|`, `^`, `~`, `<<`, `>>` 等操作。
- **Pointer & Memory**: 掌握 C 語言指標、`malloc`/`free` 以及 Linked List 操作。
- **Complexity Analysis**: 嚴格要求時間複雜度 $O(n)$ 或 $O(\log n)$。

## 開發環境
- **OS**: Linux & Windows
- **Compiler**: GCC
- **Editor**: VS Code
